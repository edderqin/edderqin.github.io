查询操作

查询部门表的所有数据

SELECT * 
FROM departments

显示每个部门的编号和名称

SELECT department_id,department_name
FROM departments;

书写SQL语句规则

SQl语言不区分大小写
SQl可以被卸载一行或者多行
关键字不能被简写也不能分行写

空值

空值是无效的，不确定的，未知的或不可预知的值
空值不是空格或者0
空值是NULL，在SQL文等价于空字符串（”），有时在显示数据时空值不显示为null显示为“

定义别名

重命名一个列 两种方式1.紧跟列名定义 2.在列名和别名间加上’AS’关键字 
如果希望在别名中包含空格或特殊的字符或者去分大小写的话，要在定义的别名上使用双引号
SELECT last_name AS name, commission_pct comm
FROM   employees;
SELECT last_name "Name", salary*12 "Annual Salary"
FROM   employees;

连接符

把列与列，列与字符连接在一起。
用 ‘||’表示 可以用来‘合成’列。

SELECT    last_name||job_id AS "Employees"
FROM     employees;

字符串与日期

字符串和日期要包含在单引号中
字符串区分大小写
日期的默认格式是  DD-MON-RR

删除重复行 DISTINCT

关键字’DISTINCT’ 删除重复行
--如果记录有多列值，想删除重复行的话，则需要使用” DISTINCT”关键字
该关键字必须写在SELECT的后面
SELECT DISTINCT department_id
FROM employees

Sqlplus工具

sqlplus工具是Oracle数据库在安装时自带的工具，以命令行窗口作为工具的窗口，并在命令行中以命名的形式加以使用

描述表结构

查看表结构：desc 表名
编辑上一次执行过的SQL语句： ed(或edit) ，执行该命令后SQL语句会出现在一个临时文件中
当编辑好SQL语句后，要保存并关闭，此时需要输入斜线加以执行
注意：临时文件中不能以分号结尾
