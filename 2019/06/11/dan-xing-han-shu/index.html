字符函数

大小写处理函数

INITCAP
首字母大写后面小写

SELECT INITCAP(‘abc def%dsad1231dawwe’)
FROM dual;

LOWER:转换成小写
UPPER:转换成大写

字符处理函数

SUNSTR
截取
——第二个参数表示从第几个开始截取 

SELECT SUBSTR(‘Hello World!’,3),SUBSTR(‘Hello World!’,3,5),SUBSTR(‘Hello World!’,-5,3)
FROM dual;

INSTR
获取字符位置
–第三个参数为正整数表示从左向右搜索；为负数表示从右数第几个字符开始并且从右向左
–第四个字符表示第几次出现的
搜索

SELECT INSTR(‘Hello World!’,’W’),INSTR(‘Hello World!’,’o’,2), INSTR(‘Hello World!’,’l’,4,2), INSTR(‘Hello World!’,’l’,-4,2)
FROM dual

TRIM
去除元素 

SELECT TRIM(‘       Hello  World!      ’),TRIM(‘k’FROM’kkkkkkokokhelloworldkkkkkkkkk’)
FROM dual

ROUND:四舍五入
TRUNC:截断

日期函数

日期型常量必须写在单引号中。’02-8月-19’
日期格式敏感，默认的日期格式是 DD-MON-RR，DD表示日，MON表示月，RR表示年。
SYSDATE函数用于返回数据库服务器端当前日期时间。
日期的数学运算符：
日期 + 天=日期
日期 - 天=日期
日期 – 日期=天
如果想加减小时，分钟，秒的话，则需要把他们换算成天 

SELECT SYSDATE+250,SYSDATE-250,SYSDATE-hire_date
FROM employees;

计算两个日期相差月份

MONTHS_BETWEEN ('01-SEP-95','11-JAN-94')

加上月份

ADD_MONTHS ('11-JAN-94',6)

下一个星期

NEXT_DAY ('01-SEP-95','FRIDAY')

上一个星期

LAST_DAY('01-FEB-95')

隐式转换

显式转换

TO_CHAR函数

——TO_CHAR的第二个参数与结果对应，即把第一个参数值按照第二个参数的要求

对日期的转换
格式参数:

  1. 必须包含在单引号中而且区分大小写。
  2. 可以包含任意的有效的日期格式。
  3. 可以使用 fm 4. 去掉多余的空格或者前导零。
  4. 与日期参数之间用逗号隔开。
基础格式
TO_CHAR(date, 'format_model')
SELECT TO_CHAR(SYSDATE,'YYYY-MM-DD DAY HH24:MI:SS AM'),TO_CHAR(SYSDATE,'FMYYYY-MM-DD DAY HH24:MI:SS AM'),TO_CHAR(SYSDATE,'YYYY"年"-MM"月"-DD"日" DAY HH24:MI:SS AM')
FROM dual;

对数字的转换

TO_CHAR(number, 'format_model')

SELECT TO_CHAR(123456.8759,'$999,999,999.99')
FROM dual;
结果为：$123,456.88
TO_NUMBER

使用 TO_NUMBER 函数将字符转换成数字:

TO_NUMBER(char[, 'format_model'])
TO_DATE

使用 TO_DATE 函数将字符转换成日期:

TO_DATE(char[, 'format_model'])

通用函数

这些函数可以用于任何数据类型，同时也适用于空值

NVL (expr1, expr2)
NVL2 (expr1, expr2, expr3)
NULLIF (expr1, expr2)
COALESCE (expr1, expr2, ..., exprn)

NVL 函数

将空值转换成一个已知的值
可以使用的数据类型有日期、字符和数字。
参数的数据类型必须一致: 

SELECT first_name,NVL(commission_pct,0)
FROM employees;

NVL2 函数

第一个参数值不是空值，则返回第二个参数值，第一个参数值是空值，则返回第三个参数值。

SELECT NVL2(commission_pct,1,9999)
FROM employees;

NULLIF函数

两个参数值不相等，则返回第一个参数值，两个参数值相等则返回NULL。

SELECT first_name,LENGTH(first_name) "expr1",
            last_name,LENGTH(last_name) "expr2",
            NULLIF(LENGTH(first_name),LENGTH(last_name)) as "结果"
FROM employees;

COALESCE函数

返回第一个不为NULL的值

SELECT COALESCE(commission_pct,salary,100)
FROM employees

嵌套函数

单行函数可以嵌套，嵌套函数的执行顺序是由内到外

SELECT last_name,
       NVL(TO_CHAR(manager_id), 'No Manager')
FROM   employees
WHERE  manager_id IS NULL;

条件表达式

在 SQL 语句中使用IF-THEN-ELSE 逻辑。
使用两种方法:
CASE 表达式
DECODE 函数

CASE 表达式

Case表达式，是可以在sql中使用if ..then..else的逻辑判断，而避免使用PL/SQL的有效方法。
——CASE表达式有两种写法，第一种写法只能进行相等性比较，第二种写法可以使用任意比较运算符。 

SELECT last_name,job_id,salary,
CASE job_id WHEN 'IT_PROG' THEN 1.10*salary
         WHEN 'ST_CLERK' THEN 1.15*salary
         WHEN 'SA_REP' THEN 1.20*salary
ELSE     salary
END AS "Salary"
FROM employees
SELECT last_name,job_id,salary,
CASE WHEN job_id LIKE '%REP%' THEN 0.001*salary
ELSE     salary
END AS "Salary"
FROM employees

DECODE 函数

语法：
decode(条件，值1，返回值1，值2，返回值2，…值n,返回值n，缺省值) [1] 

SELECT last_name,job_id,salary,
    DECODE(job_id,   'IT_PROG', 1.10*salary,
                   'ST_CLERK', 1.15*salary,
                                           'SA_REP', 1.20*salary,    
        salary
) as "Salary"
FROM employees
